RoboLib Documentation

This is the RoboLib.

It is designed to handle the hardware modules of the keplerboard and similar devices.

It consists of several parts:

  • hardware interaction - low level
    • Adc (robolib/adc.h)
    • Systick (current) (robolib/tick.h)
    • Timer (future) (robolib_future/timer.h)
    • UART0 (robolib/uart0.h)
    • UART1 (robolib/uart1.h)
    • Display (robolib/display.h)
  • hardware interaction - second level
    • Motor (future) (robolib_future/motor.h)
    • Systick (future) (robolib_future/systick.h)
    • Adc timer (robolib/adc_timer.h)
    • Display buffer (robolib/display_buffer.h)
  • helper functions
    • Delay (robolib/delay.h)
    • String (robolib/string.h)
    • Bootloader (robolib/bootloader.h)
  • declarations
    • Assembler (robolib/assembler.h)
    • Types (robolib/types.h)
  • predefined ini-files
    • Keplerboard (ini/keplerboard.ini)
    • Keplerboard2 (ini/keplerboard2.ini)
    • Keplerboard3 (ini/keplerboard3.ini)
